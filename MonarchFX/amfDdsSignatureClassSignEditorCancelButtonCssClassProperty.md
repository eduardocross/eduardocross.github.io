This property sets the CSS Class for the Signature Editor's Cancel button.

#### Syntax
<pre class="prettyprint"> **BegProp SignEditorCancelButtonCssClass Access(*Public) Type(*String)
   BegGet;  BegSet** </pre>

#### Property Values
String. A string containing the CSS Class to apply to the Cancel button on the Signature Editor. Defaults to "DdsSignCancelButton".

#### Remarks
This sets the CSS Class for the Cancel button on the Signature Editor canvas. Cascading Style Sheets allow you to set a number of styling options.
