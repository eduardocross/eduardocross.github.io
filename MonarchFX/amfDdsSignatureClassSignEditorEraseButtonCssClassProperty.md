This property sets the CSS Class for the Signature Editor's Erase button.

#### Syntax
<pre class="prettyprint"> **BegProp SignEditorEraseButtonCssClass Access(*Public) Type(*String)
   BegGet;  BegSet** </pre>

#### Property Values
String. A string containing the CSS Class to apply to the Erase button on the Signature Editor. Defaults to "DdsSignEraseButton".

#### Remarks
This sets the CSS Class for the Erase button on the Signature Editor canvas. Cascading Style Sheets allow you to set a number of styling options.
