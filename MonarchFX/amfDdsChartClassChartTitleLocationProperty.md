Gets or sets the location of the title for the chart control. 

#### Syntax
<pre class="prettyprint"> **BegProp ChartTitleLocation Access(*Public) Type(*String)
   BegGet;  BegSet** </pre>

#### Property Values
Enum. Enumeration value that defines the location of the display name for the chart. It accepts the following values:

- Left -
	   	Aligns the chart title to the left.
- Right -
	   	Aligns the chart title to the right.
- Center -
	   	Aligns the chart title to the chart's center.

#### Remarks
Sets the location for the display title for the chart.
