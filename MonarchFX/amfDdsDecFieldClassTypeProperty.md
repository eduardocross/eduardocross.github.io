Gets or sets the type of decimal field; either Zoned, Packed or Decimal.

#### Syntax
<pre class="prettyprint"> **BegProp Type Access(*Public) Type([ASNA.Monarch.WebDspF.DecType](amfDecTypeEnumeration.html))
   BegGet;  BegSet** </pre>

#### Property Values
**ASNA.Monarch.WebDspF.DecType** for the type of decimal field.

#### Remarks
To set this property at design time, click on the arrow on the right side of the property window and select one of the options shown in the drop-down box. The default type is **Zoned** .
